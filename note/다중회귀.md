# 다중회귀 (Multiple Regression)

다중회귀는 2개 이상의 [독립변수](독립변수.md)를 사용하여 [종속변수](종속변수.md)를 예측하는 [회귀분석](회귀분석.md)이다. 현실 데이터는 하나의 변수만으로 설명되기 어렵기 때문에 단순선형회귀보다 널리 활용된다.

$$\hat{y} = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_k x_k$$

각 [회귀계수](회귀계수.md) $\beta_j$는 나머지 독립변수들을 고정했을 때 $x_j$가 1단위 증가할 때의 효과인 편회귀계수(Partial Regression Coefficient)를 의미한다.

독립변수들 사이의 강한 [선형 관계](선형%20관계.md)인 다중공선성(Multicollinearity)은 [회귀계수](회귀계수.md) 추정을 불안정하게 만들며, VIF(Variance Inflation Factor)로 진단한다. 모델 적합도는 수정 [R²](R².md)와 F-통계량으로 평가한다. [OLS](OLS.md)로 계수를 추정하며 [잔차](잔차.md) 가정 검토가 필수적이다.
