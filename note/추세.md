# 추세 (Trend)

[시계열](시계열.md) 데이터에서 장기적으로 지속되는 증가 또는 감소의 방향성. 시계열의 4대 구성 요소(추세, 계절성, 순환, 불규칙) 중 하나이며, [정상성](정상성.md) 위반의 주요 원인이다.

추세의 종류:
- **결정론적 추세(Deterministic Trend)**: $X_t = \alpha + \beta t + \varepsilon_t$ — 시간의 함수로 표현 가능
- **확률론적 추세(Stochastic Trend)**: 단위근(unit root)을 가지며 무작위 보행(random walk) 형태

결정론적 추세는 회귀분석에서 시간 변수를 추가하거나 추세 제거(detrending)로 대응한다. 확률론적 추세는 [차분](차분.md)을 통해 제거한다. [ADF 검정](ADF%20검정.md)으로 어떤 형태의 추세인지 판별할 수 있다.

추세 제거 후에도 [자기상관](자기상관.md)이 남아 있다면 ARMA/ARIMA 모형으로 추가 분석한다.
