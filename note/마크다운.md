# 마크다운

## 핵심 개념
마크다운(Markdown)은 2004년 존 그루버가 개발한 경량 마크업 언어(Lightweight Markup Language)로, 일반 텍스트 형식으로 작성하여 HTML 등으로 변환할 수 있다. 읽기 쉽고 쓰기 쉬운 문법으로 문서 작성, 기술 문서화, 노트 테이킹에 널리 사용된다.

## 주요 특징
- 간결한 문법으로 제목, 목록, 링크, 이미지 표현 
	- [기초적인 마크다운 사용법](./기초적인%20마크다운%20사용법.md) 참조
- 순수 텍스트 기반으로 버전 관리 시스템과 호환성 우수
- HTML로 자동 변환 가능
- GitHub, Notion, Obsidian, Jekyll 등 다양한 플랫폼 지원
- 코드 블록과 수식 표현 지원으로 기술 문서 작성에 유용
