# 파이썬 데이터타입

## 핵심 개념
[파이썬](./파이썬.md)에서 데이터타입(Data Type)은 변수가 저장하는 값의 종류와 해당 값에 수행할 수 있는 연산을 정의한다. 이는 [데이터 타입 (통계학)](./데이터%20타입%20(통계학).md)와 유사하게 데이터를 분류하는 체계이다.

## 주요 데이터타입
### 기본 타입 (Primitive Types)
- `int`: 정수형 (예: 1, 100, -5)
- `float`: 부동소수점 실수형 (예: 3.14, -0.5)
- `str`: 문자열 (예: "Hello", 'Python')
- `bool`: 불린형 (True, False)

### 컬렉션 타입 (Collection Types)
- `list`: 순서가 있는 변경 가능한 시퀀스 (예: [1, 2, 3])
- `tuple`: 순서가 있는 변경 불가능한 시퀀스 (예: (1, 2, 3))
- `dict`: 키-값 쌍의 매핑 (예: {"name": "홍길동"})
- `set`: 중복 없는 집합 (예: {1, 2, 3})
