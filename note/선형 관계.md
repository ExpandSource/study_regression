# 선형 관계 (Linear Relationship)

선형 관계는 두 [변수 (통계학)](변수%20(통계학).md) 간의 [관계 (통계)](관계%20(통계).md)가 직선으로 표현될 수 있는 경우를 말한다. [산점도 scatter plot](산점도%20scatter%20plot.md)에서 데이터가 직선 형태로 분포할 때 선형 관계가 있다고 판단한다.

$$y = ax + b$$

[Pearson (통계)](Pearson%20(통계).md) [상관계수](상관계수.md)는 선형 관계의 강도와 방향을 측정한다. 상관계수가 +1 또는 -1에 가까울수록 강한 선형 관계를 나타낸다.

선형 관계는 [단조 관계](단조%20관계.md)의 특수한 경우이다. [비선형 관계](비선형%20관계.md)와 달리 일정한 변화율을 가지며, 회귀분석의 기본 가정이 된다.
