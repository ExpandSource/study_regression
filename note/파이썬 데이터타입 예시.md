# 파이썬 데이터타입 예시

## 핵심 개념
[파이썬 데이터타입](./파이썬%20데이터타입.md)의 실제 사용 예시를 통해 각 타입의 특성과 활용 방법을 이해한다. 타입별 선언, 연산, 메서드 사용법을 코드로 확인할 수 있다.

## 기본 타입 예시

### 정수형 (int)
```python
# 정수 선언 및 연산
age = 25
count = 100
total = age + count  # 125

# 다양한 진법 표현
binary = 0b1010      # 2진법: 10
octal = 0o12         # 8진법: 10
hexadecimal = 0xA    # 16진법: 10
```

### 실수형 (float)
```python
# 실수 연산
height = 175.5
weight = 68.3
bmi = weight / (height/100) ** 2  # 22.2

# 과학적 표기법
speed_of_light = 3e8  # 3 × 10^8
```

### 문자열 (str)
```python
# 문자열 조작
name = "Python"
greeting = f"Hello, {name}!"  # f-string
upper_name = name.upper()      # "PYTHON"
sliced = name[0:3]             # "Pyt"
```

### 불린형 (bool)
```python
# 조건 평가
is_valid = True
is_empty = False
result = (10 > 5) and is_valid  # True
```

## 컬렉션 타입 예시

### 리스트 (list)
```python
# 리스트 생성 및 조작
numbers = [1, 2, 3, 4, 5]
numbers.append(6)           # [1, 2, 3, 4, 5, 6]
numbers[0] = 10             # [10, 2, 3, 4, 5, 6]
sliced_list = numbers[1:4]  # [2, 3, 4]
```

### 튜플 (tuple)
```python
# 튜플 - 변경 불가능
coordinates = (37.5665, 126.9780)  # 서울 좌표
# coordinates[0] = 0  # TypeError 발생!
```

### 딕셔너리 (dict)
```python
# 딕셔너리 - 키-값 매핑
student = {
    "name": "홍길동",
    "age": 20,
    "major": "Statistics"
}
print(student["name"])        # "홍길동"
student["grade"] = "A+"       # 새 키 추가
```

### 집합 (set)
```python
# 집합 - 중복 제거 및 집합 연산
fruits = {"apple", "banana", "apple"}  # {"apple", "banana"}
colors = {"red", "blue", "green"}
fruits.add("orange")                   # 요소 추가

# 집합 연산
set1 = {1, 2, 3}
set2 = {2, 3, 4}
union = set1 | set2        # {1, 2, 3, 4}
intersection = set1 & set2  # {2, 3}
```

## 타입 변환 (Type Conversion)
```python
# 타입 간 변환
num_str = "123"
num_int = int(num_str)      # 123
num_float = float(num_str)  # 123.0

# 타입 확인
print(type(num_int))        # <class 'int'>
print(isinstance(num_int, int))  # True
```
