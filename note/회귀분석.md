# 회귀분석 (Regression Analysis)

회귀분석은 [독립변수](독립변수.md)와 [종속변수](종속변수.md) 사이의 관계를 수학적 모델로 표현하여 예측과 설명을 수행하는 통계 기법이다. 프랜시스 골턴(Francis Galton)이 키의 "평균으로의 회귀" 현상을 연구하면서 명명되었다.

가장 기본적인 형태인 단순선형회귀(Simple Linear Regression)는 한 개의 독립변수로 종속변수를 예측한다:

$$\hat{y} = \beta_0 + \beta_1 x$$

여기서 $\beta_0$는 [절편](절편.md), $\beta_1$은 [회귀계수](회귀계수.md)이다. 회귀선은 [잔차](잔차.md)의 제곱합을 최소화하는 [OLS](OLS.md) 방법으로 추정한다.

모델의 적합도는 [R²](R².md)로 평가하며, 독립변수가 여러 개인 경우 [다중회귀](다중회귀.md)로 확장된다. [상관계수](상관계수.md)가 높다고 반드시 좋은 모델이 되는 것은 아니며, 잔차 분석이 필수적이다.
