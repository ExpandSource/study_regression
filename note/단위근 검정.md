# 단위근 검정 (Unit Root Test)

[시계열](시계열.md) 데이터에 단위근(unit root)이 존재하는지 통계적으로 검정하는 방법. 단위근이 있는 시계열은 비[정상성](정상성.md)을 가지므로, 분석 전 반드시 확인해야 한다.

단위근이란 특성 방정식의 근이 1인 경우를 의미한다. 단순 AR(1) 모형 $X_t = \rho X_{t-1} + \varepsilon_t$에서:
- $|\rho| < 1$: 정상 시계열
- $\rho = 1$: 단위근 존재 → 비정상 시계열 (무작위 보행)

대표적인 단위근 검정 방법:
- **[ADF 검정](ADF%20검정.md)** (Augmented Dickey-Fuller): 가장 널리 사용
- **KPSS 검정**: 귀무가설이 정상성 (ADF와 반대)
- **PP 검정** (Phillips-Perron): ADF의 비모수적 대안

단위근이 확인되면 [차분](차분.md)을 통해 정상 시계열로 변환한 후 분석을 진행한다. [자기상관](자기상관.md)의 구조에 따라 적절한 검정 방법을 선택한다.
