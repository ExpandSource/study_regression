# 히스토그램

## 핵심 개념
히스토그램(Histogram)은 연속형 데이터의 분포를 시각화하는 [기술통계](./기술통계.md) 그래프이다. 데이터를 구간(Bin)으로 나누고 각 구간의 빈도(Frequency)를 막대 높이로 표현한다. [Matplotlib](./Matplotlib.md)에서 `plt.hist()`로 생성한다.

## 구성 요소
- **구간(Bin)**: 데이터를 나누는 범위 (예: 0-10, 10-20, ...)
- **빈도(Frequency)**: 각 구간에 속하는 데이터 개수
- **막대**: 연속된 막대로 표현 (막대 그래프와 달리 간격 없음)

## 특징
- 데이터 분포의 형태 파악 (정규분포, 왜도, 첨도)
- 구간 수(bins)에 따라 분포 해석이 달라짐
- 이상치 및 다봉분포(Multimodal) 탐지 가능

## 막대 그래프와의 차이
| 히스토그램 | 막대 그래프 |
|-----------|-----------|
| 연속형 데이터 | 범주형 데이터 |
| 막대 간 간격 없음 | 막대 간 간격 있음 |
