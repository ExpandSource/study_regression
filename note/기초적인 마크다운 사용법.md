# 기초적인 마크다운 사용법

## 핵심 개념
[마크다운](./마크다운.md)의 기본 문법(Basic Syntax)은 제목, 강조, 목록, 링크, 코드 등을 일반 텍스트로 표현하는 규칙이다. HTML을 몰라도 구조화된 문서를 쉽게 작성할 수 있다.

## 제목 (Headings)
```markdown
# 제목 1 (H1)
## 제목 2 (H2)
### 제목 3 (H3)
```

## 강조 (Emphasis)
```markdown
**굵게** 또는 __굵게__ (Bold)
*기울임* 또는 _기울임_ (Italic)
~~취소선~~ (Strikethrough)
```

## 목록 (Lists)
```markdown
- 순서 없는 목록 (Unordered List)
- 항목 2
  - 하위 항목

1. 순서 있는 목록 (Ordered List)
2. 항목 2
```

## 링크와 이미지 (Links & Images)
```markdown
[링크 텍스트](https://example.com)
![이미지 설명](image-url.png)
```

## 코드 (Code)
```markdown
인라인 코드: `print("Hello")`

코드 블록:
\`\`\`python
def hello():
    print("Hello World")
\`\`\`
```

## 인용 (Blockquotes)
```markdown
> 인용문입니다.
> 여러 줄도 가능합니다.
```

## 수평선 (Horizontal Rule)
```markdown
---
***
```

## 표 (Tables)
```markdown
| 제목1 | 제목2 |
|------|------|
| 내용1 | 내용2 |
```
