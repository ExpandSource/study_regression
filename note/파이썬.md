# 파이썬

## 핵심 개념
파이썬(Python)은 1991년 귀도 반 로섬(Guido van Rossum)이 개발한 고급 프로그래밍 언어(High-level Programming Language)로, 간결하고 읽기 쉬운 문법을 특징으로 한다. 인터프리터(Interpreter) 방식으로 동작하며, [Colab](./Colab.md)과 같은 환경에서 즉시 실행 가능하다.

## 주요 특징
- **가독성 높은 문법**: 들여쓰기로 코드 블록을 구분하는 직관적 구조
- **동적 타입 언어**: 실행 시점에 변수 타입 결정 ([파이썬 데이터타입](./파이썬%20데이터타입.md) 참조)
- **다중 패러다임**: 절차적, 객체지향, 함수형 프로그래밍 지원
- **풍부한 생태계**: [NumPy](./NumPy.md), [Pandas](./Pandas.md), [Matplotlib](./Matplotlib.md) 등 라이브러리
- **크로스 플랫폼**: Windows, macOS, Linux 지원
- **활발한 커뮤니티**: 오픈소스 기반의 광범위한 개발자 생태계
