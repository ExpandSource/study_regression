# 히트맵 (Heatmap)

히트맵은 행렬 형태의 데이터를 색상의 강도로 표현하는 시각화 기법이다. 값의 크기를 색상 그라데이션(color gradient)으로 나타내어 패턴을 직관적으로 파악할 수 있다.

[통계학](통계학.md)에서는 주로 [상관행렬](상관행렬.md)이나 [공분산](공분산.md) 행렬을 시각화할 때 사용한다. 여러 [변수 (통계학)](변수%20(통계학).md) 간의 [관계 (통계)](관계%20(통계).md)를 한눈에 비교할 수 있어 탐색적 데이터 분석에 유용하다.

[seaborn](seaborn.md)의 `sns.heatmap()` 함수로 쉽게 생성할 수 있으며, `annot=True` 옵션으로 셀에 수치를 표시하고 `cmap` 옵션으로 색상표를 지정한다.
