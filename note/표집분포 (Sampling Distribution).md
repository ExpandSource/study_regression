# 표집분포 (Sampling Distribution)

[통계량 (Statistic)](통계량%20(Statistic).md)의 [확률분포](확률분포.md). 동일한 [모집단 (Population)](모집단%20(Population).md)에서 같은 크기의 [표본 (Sample)](표본%20(Sample).md)을 반복적으로 추출할 때, 각 표본으로부터 계산된 통계량들이 따르는 분포를 의미한다.

예를 들어, [표본평균 (Sample Mean)](표본평균%20(Sample%20Mean).md)의 표집분포는 여러 표본에서 계산된 평균들의 분포이다. [중심극한정리 (Central Limit Theorem, CLT)](중심극한정리%20(Central%20Limit%20Theorem,%20CLT).md)에 의하면, 표본 크기가 충분히 크면 표본평균의 표집분포는 근사적으로 정규분포를 따른다.

표집분포의 표준편차를 [표준오차 (Standard Error, SE)](표준오차%20(Standard%20Error,%20SE).md)라 하며, 이는 통계량의 변동성을 나타낸다. 표본평균의 경우 $SE = \frac{\sigma}{\sqrt{n}}$로 표본 크기가 클수록 작아진다.

표집분포는 [신뢰구간 (Confidence Interval, CI)](신뢰구간%20(Confidence%20Interval,%20CI).md) 구성과 가설검정의 이론적 기반이 되며, [모수 (Parameter)](모수%20(Parameter).md) 추정의 불확실성을 정량화하는 데 핵심적인 역할을 한다.
