# Colab

## 핵심 개념
구글 코랩(Google Colaboratory, Colab)은 브라우저에서 [파이썬](./파이썬.md) 코드를 작성하고 실행할 수 있는 클라우드 기반 주피터 노트북(Jupyter Notebook) 환경이다. 별도 설치 없이 무료로 GPU/TPU를 활용한 머신러닝과 데이터 분석이 가능하다.

## 주요 특징
- 클라우드 기반으로 언제 어디서나 접근 가능
- GPU/TPU 무료 제공 (시간 제한 있음)
- Google Drive 연동으로 파일 관리 용이
- 실시간 협업 가능
- 데이터 분석, 머신러닝 교육에 최적화
