# 절편 (Intercept)

절편은 [회귀분석](회귀분석.md)에서 [독립변수](독립변수.md) 값이 0일 때 [종속변수](종속변수.md)의 예측값이다. 회귀식 $\hat{y} = \beta_0 + \beta_1 x$에서 $\beta_0$에 해당하며, 회귀선이 y축과 만나는 지점이다.

$$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$$

절편은 [회귀계수](회귀계수.md)와 함께 [OLS](OLS.md)로 추정된다. 독립변수가 0인 상황이 현실적으로 의미 없는 경우(예: 키가 0인 사람)에는 절편 자체를 해석하지 않는 것이 일반적이다.

절편을 제거한 원점 통과 회귀(Regression through the origin)를 사용하는 경우도 있으나, 이는 모델 가정을 강하게 제약하므로 이론적 근거가 있을 때만 적용한다. [잔차](잔차.md)의 평균이 0이 되는 성질은 절편이 포함된 회귀모델에서만 보장된다.
