# 경험적 분포

경험적 분포(Empirical Distribution)는 실제로 관측된 데이터로부터 얻어지는 [분포](./분포%20(통계학).md)이다. [이론적 분포](./이론적%20분포.md)와 달리 수학적 공식이 아닌 실제 [표본](./표본%20(통계학).md) 데이터에 기반한다.

경험적 누적분포함수(Empirical Cumulative Distribution Function, ECDF)는 각 값 이하인 데이터의 비율로 정의된다.

$$\hat{F}_n(x) = \frac{1}{n}\sum_{i=1}^{n} \mathbf{1}_{X_i \le x}$$

[히스토그램](./히스토그램.md)은 경험적 분포를 시각화하는 가장 기본적인 방법이다. 커널 밀도 추정(KDE)을 통해 부드러운 [확률밀도](./확률밀도.md) 곡선을 추정할 수 있다.

표본 크기가 커질수록 경험적 분포는 모집단의 실제 분포에 수렴한다(Glivenko-Cantelli 정리). [기술통계](./기술통계.md)에서 데이터 탐색의 첫 단계로 활용된다.

경험적 분포와 이론적 분포를 비교하여 데이터가 특정 분포를 따르는지 검정할 수 있다([Q-Q plot](./Q-Q%20plot.md), [Shapiro-Wilk 검정](./Shapiro-Wilk%20검정.md)).
