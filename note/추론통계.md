# 추론통계

## 핵심 개념
추론통계(Inferential Statistics)는 [통계학](./통계학.md)의 한 분야로, [표본](./표본%20(통계학).md)의 정보를 바탕으로 [모집단](./모집단%20(통계학).md)의 특성을 추정하고 가설을 검정하는 방법론이다. [기술통계](./기술통계.md)가 데이터를 요약한다면, 추론통계는 불확실성 하에서 결론을 도출한다.

## 주요 분야
### 추정(Estimation)
- **점추정(Point Estimation)**: 모수를 하나의 값으로 추정 (예: $\hat{\mu} = \bar{x}$)
- **구간추정(Interval Estimation)**: 신뢰구간으로 모수 범위 추정 (예: 95% 신뢰구간)

### 가설검정(Hypothesis Testing)
- **귀무가설(Null Hypothesis, $H_0$)**: 기존 주장, 효과 없음
- **대립가설(Alternative Hypothesis, $H_1$)**: 새로운 주장, 효과 있음
- **유의수준(Significance Level, $\alpha$)**: 1종 오류의 허용 확률
