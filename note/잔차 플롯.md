# 잔차 플롯 (Residual Plot)

회귀모델의 [잔차](잔차.md)(residuals)를 시각화한 그래프. 주로 예측값(fitted values)을 x축으로, 잔차를 y축으로 나타내며, 회귀모델의 가정을 진단하는 데 사용된다.

잘 적합된 모델의 잔차 플롯은 다음 조건을 만족해야 한다:
- 잔차가 0 주변에 무작위로 분포 (선형성 확인)
- 잔차의 퍼짐이 일정 ([등분산성](등분산성.md) 확인)
- 패턴이나 구조가 없어야 함 (독립성 확인)

잔차 플롯에서 특정 패턴이 관찰되면 모델의 문제를 나타낸다. 깔때기 형태는 이분산성(heteroscedasticity), 곡선 형태는 비선형 관계를 의미한다. [Q-Q plot](Q-Q%20plot.md)과 함께 [정규성](정상성.md) 검토에도 활용된다.

Python에서는 `matplotlib`이나 `seaborn`으로 그릴 수 있으며, `statsmodels`의 `plot_regress_exog()` 함수도 활용된다.
