# 독립변수 (Independent Variable)

독립변수는 [회귀분석](회귀분석.md)에서 [종속변수](종속변수.md)를 설명하거나 예측하기 위해 사용하는 변수이다. 설명변수(Explanatory Variable), 예측변수(Predictor Variable), 특성(Feature)이라고도 불린다.

[변수 (통계학)](변수%20(통계학).md)의 분류에서 원인에 해당하는 쪽으로, 연구자가 조작하거나 관찰하는 변수이다. 단순선형회귀에서는 독립변수가 1개이며, [다중회귀](다중회귀.md)에서는 2개 이상의 독립변수를 사용한다.

$$\hat{y} = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_k x_k$$

독립변수들 사이에 강한 [선형 관계](선형%20관계.md)가 존재하면 다중공선성(Multicollinearity) 문제가 발생해 [회귀계수](회귀계수.md) 추정이 불안정해진다. 독립변수 선택은 모델 성능에 직접적인 영향을 미친다.
