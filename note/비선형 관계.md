# 비선형 관계 (Nonlinear Relationship)

비선형 관계는 두 [변수 (통계학)](변수%20(통계학).md) 간의 [관계 (통계)](관계%20(통계).md)가 곡선 형태로 나타나는 경우를 말한다. [산점도 scatter plot](산점도%20scatter%20plot.md)에서 데이터가 직선이 아닌 곡선 패턴을 보인다.

지수 관계(exponential), 로그 관계(logarithmic), 다항 관계(polynomial), U자형 관계 등이 대표적인 비선형 관계 유형이다.

[Pearson (통계)](Pearson%20(통계).md) [상관계수](상관계수.md)는 비선형 관계를 제대로 포착하지 못한다. 강한 곡선 관계가 있어도 상관계수가 0에 가깝게 나올 수 있어 주의가 필요하다.

비선형 관계 분석에는 [Spearman](Spearman.md) 상관계수나 다항회귀, 비선형 회귀 등의 방법을 사용한다.
