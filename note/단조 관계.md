# 단조 관계 (Monotonic Relationship)

단조 관계는 한 [변수 (통계학)](변수%20(통계학).md)가 증가할 때 다른 변수가 일관되게 증가하거나(단조 증가, monotonically increasing) 감소하는(단조 감소, monotonically decreasing) [관계 (통계)](관계%20(통계).md)를 말한다.

[선형 관계](선형%20관계.md)는 단조 관계의 특수한 경우이다. 단조 관계는 직선일 필요 없이 방향성만 일정하면 된다. 예를 들어 로그 함수나 제곱근 함수도 단조 관계이다.

[Spearman](Spearman.md) [상관계수](상관계수.md)는 선형성이 아닌 단조성을 측정한다. 따라서 [비선형 관계](비선형%20관계.md)이더라도 단조적이면 Spearman 상관계수가 높게 나타난다.

[산점도 scatter plot](산점도%20scatter%20plot.md)에서 곡선 형태이더라도 한 방향으로 꾸준히 증가/감소하면 단조 관계가 있다고 판단한다.
