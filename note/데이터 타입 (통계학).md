# 데이터 타입 (통계학)

## 핵심 개념
[통계학](./통계학.md)에서 데이터(Data)는 측정 수준(Level of Measurement)에 따라 질적 데이터(Qualitative)와 양적 데이터(Quantitative)로 분류된다. 이는 적절한 통계 분석 방법을 선택하는 기준이 되며, [파이썬 데이터타입](./파이썬%20데이터타입.md)과도 연관된다.

## 데이터 분류
### 질적 데이터 (Categorical Data)
- **명목형(Nominal)**: 순서 없는 범주 (예: 성별, 혈액형)
- **순서형(Ordinal)**: 순서 있는 범주 (예: 학년, 만족도)

### 양적 데이터 (Numerical Data)
- **이산형(Discrete)**: 셀 수 있는 정수값 (예: 학생 수, 판매량)
- **연속형(Continuous)**: 측정 가능한 실수값 (예: 키, 체중, 온도)
