# 회귀계수 (Regression Coefficient)

회귀계수는 [회귀분석](회귀분석.md)에서 [독립변수](독립변수.md)가 1단위 증가할 때 [종속변수](종속변수.md)가 평균적으로 변화하는 양을 나타내는 수치이다. 기울기(slope)라고도 한다.

$$\hat{y} = \beta_0 + \beta_1 x \quad \Rightarrow \quad \beta_1 = \frac{\text{Cov}(X,Y)}{\text{Var}(X)}$$

[OLS](OLS.md)로 추정된 회귀계수 $\hat{\beta}_1$은 [잔차](잔차.md) 제곱합을 최소화하는 값이다. 부호(양수/음수)는 관계의 방향을, 크기는 영향력의 강도를 나타낸다.

회귀계수의 통계적 유의성은 t-검정([t-test](t-test.md))으로 확인하며, [p-value](p-value.md)가 [유의수준](유의수준.md)보다 작으면 해당 독립변수가 유의미하게 종속변수에 영향을 준다고 판단한다. [다중회귀](다중회귀.md)에서는 각 독립변수의 회귀계수를 편회귀계수(Partial Regression Coefficient)라 한다.
