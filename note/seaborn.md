# Seaborn

Seaborn은 [Matplotlib](Matplotlib.md)을 기반으로 한 [파이썬](파이썬.md) 데이터 시각화 라이브러리이다. 통계적 그래픽(statistical graphics)을 쉽게 그릴 수 있도록 고수준 인터페이스(high-level interface)를 제공한다.

[DataFrame](DataFrame.md)과의 통합이 뛰어나며, [Pandas](Pandas.md) 데이터를 직접 입력으로 사용할 수 있다. 기본 스타일이 미적으로 세련되어 있어 최소한의 코드로 출판 품질의 시각화가 가능하다.

[히스토그램](히스토그램.md), [Box Plot](Box%20Plot.md), [산점도](산점도%20scatter%20plot.md), 히트맵(heatmap) 등 [통계학](통계학.md)적 분석에 필요한 다양한 플롯 유형을 지원한다. `hue`, `col`, `row` 파라미터를 통해 다차원 데이터를 손쉽게 시각화할 수 있다.
