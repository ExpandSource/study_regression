# 사분위수

## 핵심 개념
사분위수(Quartile)는 데이터를 크기순으로 정렬했을 때 4등분하는 위치의 값이다. [기술통계](./기술통계.md)에서 데이터의 분포와 산포도를 파악하는 데 사용되며, [Box Plot](./Box%20Plot.md)의 기본 구성 요소이다.

## 종류
- **Q1 (제1사분위수)**: 하위 25% 지점, 중앙값의 왼쪽 절반의 중앙값
- **Q2 (제2사분위수)**: 50% 지점, 중앙값(Median)과 동일
- **Q3 (제3사분위수)**: 상위 25% 지점, 중앙값의 오른쪽 절반의 중앙값

## 사분위 범위 (IQR)
$$IQR = Q3 - Q1$$

> IQR은 중앙 50%의 데이터가 분포하는 범위로, 이상치에 덜 민감한 산포도 측정치이다.

## 5-Number Summary
최솟값, Q1, Q2(중앙값), Q3, 최댓값으로 데이터 분포를 요약한다.
